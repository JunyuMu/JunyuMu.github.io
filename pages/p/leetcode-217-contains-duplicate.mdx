```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        return len(set(nums)) != len(nums)
```

利用 set 去重的特性，对比去重后的数组长度与原数组长度是否相等，很优雅，但速度应该会慢一些。
